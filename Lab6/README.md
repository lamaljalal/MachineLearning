# Ecommerce Customers - Linear Regression

This project uses **Linear Regression** to predict **Yearly Amount Spent** by customers using Python.

## Dataset Features
- Avg. Session Length
- Time on App
- Time on Website
- Length of Membership

## Target
- Yearly Amount Spent

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
- Load and explore the dataset
- Clean the data
- Select features and target
- Split data into training and testing sets
- Train a Linear Regression model
- Make predictions
- Evaluate performance using MAE, MSE, RMSE, and R²
- Visualize results

## Key Findings
- Length of Membership is the strongest predictor.
- Time on App has more impact than Time on Website.
- The model performs well in predicting yearly customer spending.
